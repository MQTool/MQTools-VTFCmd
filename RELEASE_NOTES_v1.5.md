# MQTools-VTFCmd v1.5

集成版 GUI 可执行文件（Windows）。

## 运行要求
- `vtfcmd.exe` 可用（加入 PATH 或与 exe 同目录）
- 建议安装 `ImageMagick (magick.exe)`（用于通道分析与发光贴图生成）

## 使用
- 双击运行 `MQTools-VTFCmd v1.5.exe`
- 详细说明见仓库内 `VTF-Material-Tool-Integrated/README.md`
